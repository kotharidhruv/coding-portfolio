Projects

ISS Tracker/Meteor Info

Date: May 2023

Course: Personal Project

Description: The objective of this project was to create an application which uses the NASA APIs to get information about where the ISS is and some information about different meteors near Earth. I wrote this application in React Native. I utilized NASAs public APIs to retrieve the information I wanted. The application allows users to get the real time location of the ISS and see it on an interactive map. In the meteors page, users can see different information about different meteors.

Role: Lead Developer

Learnings: Improved my React Native skills and learned about APIs and how they work

Challenges: Understanding where the data of the API is and retrieving it to display to the user

Skills Demonstrated: Technical knowledge in React Native, APIs, and problem solving

SwapZone

Date: May 2024

Course: Computer Science Essentials

Description: SwapZone is one of my group projects developed in my Computer Science Essentials class (CSE). Using Google's Firebase service, we created a platform for item exchanges. Users can sign up, log in, post items, and browse listings. It's a simple yet effective solution for facilitating item swaps within our community.

Role: Team Lead & Back End Developer

Learnings: Collaboration with teams, database/data management, and technical backend development skills

Challenges: Optimizing the database structure to minimize data redundancy

Skills Demonstrated: Javascript proficiency, user-centric design, and problem solving
